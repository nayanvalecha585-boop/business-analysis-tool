# business-analysis-tool
An interactive, AI-powered data science assistant built with Streamlit, Pandas, and Plotly. Powered by the Gemini API, it automates data cleaning, generates complex visualizations, and delivers intelligent business insights directly from your data files.
# 📊 Automated Business Analysis Tool

An AI-powered web app that transforms raw CSV data into 
professional business insights instantly.

## 🔗 Live Demo
[Click here to try the app](https://business-analysis-tool-vnaw4mhs8szbmga4fsz2ft.streamlit.app/)

## ✨ Features
- Upload any business CSV file
- Auto-generates statistical summary
- 3 interactive charts — Scatter, Histogram, Box Plot
- AI-written executive insights powered by Gemini API

## 🛠️ Built With
- Python
- Streamlit
- Pandas
- Plotly
- Google Gemini API

## 🚀 How to Run Locally
pip install -r requirements.txt
streamlit run app.py
